# ConsumptionNotes

App for electricity/natural gas consumption statistics <br/>
Electricity works only with day/night mode <br/>

## Technologies
- .NET 8
- Avalonia UI
- LiveCharts2
- SQLite
- Community Toolkit Mvvm
